# Training clone Figma ด้วย vue & tailwind
This project is a frontend UI clone based on a Figma design, built to practice translating design into responsive web interfaces.

## 🚀 Features
- Pixel-perfect UI based on Figma design
- Responsive layout (mobile / tablet / desktop)
- Reusable Vue components
- Clean and structured layout

## 🛠 Tech Stack
- Vue.js
- Tailwind CSS
- Vite

## 🎯 Purpose
This project focuses on improving frontend development skills:
- Converting Figma design into real UI
- Working with modern CSS frameworks (Tailwind)
- Building reusable components in Vue
